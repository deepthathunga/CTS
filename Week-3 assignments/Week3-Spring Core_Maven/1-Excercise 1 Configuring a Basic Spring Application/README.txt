A simple Spring Framework example project that loads a bean using XML configuration and prints a message to the console.

## 📦 Project Structure
```
src/main/java/com/example/HelloWorld.java
src/main/java/com/example/MainApp.java
src/main/resources/Beans.xml
pom.xml
```

## ▶️ How to Build and Run
1. Import the project into Eclipse (File → Import → Maven → Existing Maven Projects)
2. Wait for Maven to download dependencies
3. Open `MainApp.java`
4. Right-click → Run As → Java Application

## ✅ Expected Output
```
Your Message: Hello Spring!
```

## ⚙ Technologies Used
- Java
- Spring Framework (spring-context)
- Maven
